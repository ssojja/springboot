[ 메인 & 상품 ] 파트

1번 헤더는 총 3행으로 이뤄져있고 1행과 2행은 마이페이지, 로그인, 로고, 상품 검색, 브랜드 버튼 등 클릭 시, 해당하는 화면으로 이동합니다.
3행의 상품 카테고리 탭 메뉴 2뎁스로 대분류와 중분류 구성되어져 있고 호버 시 메뉴 표시, 클릭 시 해당 카테고리 상품목록으로 이동합니다.

2번 메인슬라이더는 3열 캐러셀로 구성되어 있고, 5초 간격으로 자동 슬라이드 전환 및 클릭 시 연결된 이벤트/상품 페이지로 이동합니다.

3번 랜덤 상품 진열 그리드는 카드 호버 시 그림자 효과, 클릭 시 상품 상세 페이지로 이동합니다.

4번 브랜드 별 로고는 클릭 시 브랜드 페이지로 이동합니다.

5번 이벤트/공지사항 배너 클릭 시 게시판으로 이동합니다.

6번 키워드 별 상품 진열 그리드는 카테고리를 클릭 시 해당 카드가 진열되어 지고 카드 클릭 시 상품 페이지로 이동합니다.

7번 푸터로 구성되어있습니다.
